jQuery Slider Plugin

The following are examples of how to use the jQuery plugin.

  1. Slider with all options enabled.
  2. Slider with navigation arrows only.
  3. Slider with navigation indicators only.

To eneable this add slider.css to the head of your page and/or add the SASS file slider.scss to your main SASS file.

Then add the slider.js file to where your JavaScript is loaded. This plugin requires that jQuery already be loaded for fallback.

To initialize the plugin add the following to your page below jQuery and the slider.js file.

                                
                                      $('#replace-with-slider-id').slider({
                                          ssliderArrows: true,
                                          sliderNav: true,
                                          sliderCountDown: true,
                                          sliderPauseBtn: true
                                      });

You can change the options within the options object based on the options you want.

To see the options available look at the demo pages above.

The CSS can be overided to fit your needs. Just place it in a seperate CSS or SASS file so updates don't override your changes.

Default Options:

These are the default options that can be used for the plugin passed in to the options object.

  • autoScroll: True or false. Default is set to true.
  • autoScrollTime: Time in mili-seconds. Default is 10 seconds.
  • direction: Left or right. Default is left.
  • sliderArrows: Optional property, either true or false. Default is set to false.
  • sliderCountDown: Optional property, either true or false. Default is set to false.
  • sliderNav: Optional property, either true or false. Default is set to false.
  • sliderNavType: Optional property if sliderNav is set to true.
  • sliderPauseBtn: Optional property, either true or false. Default is set to true.
  • slideDuration: The speed of the animation.

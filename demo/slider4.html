Slider With Navigation Indicators Only

(Also a longer auto scroll time used and bullets used for the navigation indicators. )

This is a basic slider with the navigation displayed using bullets and a longer duration in between slides set.

The plugin can be initialized this way by the following:

$( '#your-slider-id' ).slider({
autoScrollTime: 20000,
navigation: true,
navigationOptions: {
indicators: true
}
});

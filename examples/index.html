jQuery Slider Plugin

The following are examples of how to use the jQuery plugin.

  1. Slider with no options enabled, just the default settings.
  2. Slider with all navigation options enabled.
  3. Slider with navigation arrows only.
  4. Slider with navigation indicators only.

To eneable this add slider.css to the head of your page and/or add the SASS file slider.scss to your main SASS file.

Then add the slider.js file to where your JavaScript is loaded. This plugin requires that jQuery already be loaded for fallback.

To initialize the plugin add the following to your page below jQuery and the slider.js file. Options that can be used can be found listed below or on the example pages.

1
   $( '#your-slider-id' ).slider();

You can change the options within the options object based on the options you want.

To see the options available look at the demo pages above.

The CSS can be overided to fit your needs. Just place it in a seperate CSS or SASS file so updates don't override your changes.

Default Options:

These are the default options that can be used for the plugin passed in to the options object.

  • autoScroll: True or false. Default is set to true for slides to automatically change.
  • autoScrollTime: The time in between each slide change.
  • duration: Specifies the number of milliseconds a transition animation should take to complete. In other words the speed of the animation.
  • navigation: Weather or not any navigation options will show.
  • navigationOptions: An object that contains all the options for navigation. All options are a boolean true/false value.
       - arrows: Indicates weather or not left or right arrows are showing.
       - countDown: This will display a countdown of how much time for a prarticular slide is left.
       - indicators: This will show idicators that show what slide is current and allow the user to chose a particular slide.
       - pauseBtn: A button that allows the user to pause the slide show will be displayed.
  • pauseOnHover: Will pause the slide show when a user hovers over an image.

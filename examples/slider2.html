This is a basic slider with all navigation options included.

You can customize the location of elements such as the timer or navigation, or other aspects of the slider by overriding the CSS.

The basic version of the plugin can be initialzed by using the following:

                            
                                  $('#slider2').slider({
                                      navigation: true,
                                      navigationOptions: {
                                          arrows: true,
                                          countDown: true,
                                          indicators: true,                               
                                          pauseBtn: true
                                      },
                                      pauseOnHover: true                         
                                  });
